# Preprint build (CytAssist breast SSL demo)

This folder is generated by `reports/preprint_breast_ssl/build_preprint.py`. It assembles a bioRxiv-style manuscript (LaTeX + PDF + Markdown) using existing demo outputs.

## Contents
- `manuscript.tex`, `manuscript.pdf`, `manuscript.md`
- `figures/` (copied from `results/figures/`)
- `references.bib`
- `build_preprint.py` (re-runs the build)

## Rebuild
From repo root:
```bash
python reports/preprint_breast_ssl/build_preprint.py
```
If `pdflatex` is available on your PATH, the PDF will be compiled; otherwise only the .tex and .md are produced.

## Expected inputs
- `data/processed/breast_cytassist_ffpe.h5ad`
- `data/processed/breast_cytassist_ffpe_radius_graph.pt`
- `results/run_breast_ssl/config_used.yaml`
- Figures in `results/figures/`:
  - `breast_total_counts_hires_cropped.png`
  - `breast_in_tissue_lowres_cropped.png`
  - `breast_radius_graph_spots_only.png`
  - `breast_umap_leiden.png`
  - `breast_spatial_leiden_lowres_cropped.png`

The build will error if these are missing.
